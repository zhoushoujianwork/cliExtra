┌─────────────────────────────────────────────────────────────────────────────┐
│                           Key Features                                      │
└─────────────────────────────────────────────────────────────────────────────┘

• Auto Instance ID Generation: cliExtra_project_timestamp_random
• Unified Workspace Management: All instances in system workspace
• Session Persistence: tmux-based session management
• Real-time Monitoring: Live instance output and logs
• Cross-project Collaboration: Instances from different projects in same namespace
• Role-based AI: Specialized AI assistants for different engineering roles
• Message Broadcasting: Instance-to-instance communication
• Context Preservation: Historical conversation and context loading
• Tool Integration: Git, DingTalk, and custom tools
• Global Availability: System-wide command access
